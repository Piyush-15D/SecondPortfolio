HVAC Prototype

I developed an HVAC prototype system using two ESP32 boards and two CAN transceivers(MCP 2515, TJA1050) to implement the CAN (Controller Area Network) protocol for communication between nodes. One ESP32 node collects temperature and humidity data using DHT11 sensor and sends the data over the CAN bus. The second ESP32 receives this data and controls actuators like motors to simulate HVAC functions like fan or vent control.
